Hugo Quick Start官网

setup1: Mac OS安装hugo

brew install hugo

setup2: 创建站点

hugo new site myblog

setup3: 添加主题

cd myblog
git init
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
echo theme = \"ananke\" >> config.toml

setup4: 启动hugo server


hogo server -D

setup5：部署到github

  1. github 新建一个以 <用户名>.github.io的仓库
  2. 发布

rm -rf public
git submodule add -b master git@github.com:hiluyu/hiluyu.github.io.git public
cd public
git add .
git commit -m '[update]'
git push

遇到的问题：

在git submodle add 创建子模块命令报错 清楚缓存后重新执行就可以了

git rm -r --cached   public
