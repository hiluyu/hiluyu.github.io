Hugo Quick Start官方文档

setup1: Mac OS安装hugo

brew install hugo

setup2: 创建站点

hugo new site myblog

setup3: 添加主题

cd myblog
git init
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
echo theme = \"ananke\" >> config.toml

setup4: 启动hugo server


hogo server -D

setup5：部署到github

在github 新建一个以 <用户名>.github.io的仓库


# public中的文件就是要部署到GitHub仓库的博客代码
rm -rf public
# 重新生成新的pubilc文件
hugo —D
cd public/
git init
git add .
git commit -m '[update]'
git remote add origin https://hiluyu/hiluyu.github.io.git
git push

我遇到的问题：

如何更新博客？

#修改后再次更新public下的文件
hugo -D
cd public/
git add .
git commit -m '[update]'
git push

下载hugo主题慢？

#方法：通过代理下载  具体的端口需要查看代理软件上proxy端口号，我这里为7890
#设置
git config --global https.proxy http://127.0.0.1:7890
git config --global https.proxy https://127.0.0.1:7890
#取消设置
git config --global --unset http.proxy
git config --global --unset https.proxy
